Web Technologies

HTML

HTML, or HyperText Markup Language, is the foundational language for creating web pages and defining their structure. It is not a programming language but a markup language that uses a system of tags and attributes to organize and present content within a web browser.

CSS

CSS, which stands for Cascading Style Sheets, is a styling language used to describe the presentation of a document written in a markup language, most commonly HTML. It defines how HTML elements are to be displayed on various media, such as screens, paper, or other devices.

JavaScript

JavaScript is a high-level, multi-paradigm programming language primarily known for enabling interactive and dynamic content on websites. It is one of the core technologies of the World Wide Web, alongside HTML (which provides structure) and CSS (which provides styling).

Ruby

Ruby is an object-oriented, open-source programming language known for its elegant and readable syntax, making it easy for developers to write and understand code quickly. It is commonly used to build dynamic, full-stack web applications, especially when paired with the popular Ruby on Rails framework, which simplifies the development process by providing a structured approach and pre-built tools for tasks like database management and rendering HTML.
